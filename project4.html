M1neral

  • Role: Full Stack Developer
  • Tech Stack: React, Node, MongoDB, Express, Elastic Search, Azure jobs, SQL, Mapbox

Description

M1NERAL is a property platform designed to provide comprehensive data on locations containing valuable resources such as oil, gas, water, and other minerals. The platform serves as a vital tool for property owners, buyers, and resource managers, offering detailed insights into property details, ownership, contacts, and mineral information.

Key Features

  • Resource Data Integration: Aggregates and displays data on properties with mineral resources, enabling users to make informed decisions.
  • Property Management: Detailed property profiles that include ownership history, mineral rights, and associated contacts.
  • GraphQL API Integration: Utilizes GraphQL for efficient data fetching, providing flexibility in querying and managing complex data sets.
  • Map Visualization: Integrated Mapbox to display geographical layers and wells on an interactive map, enhancing spatial understanding and data visualization.
  • Multitenancy Support: Implemented multitenancy, allowing multiple organizations to use the platform with isolated data and configurations, ensuring data privacy and customized user experiences.
