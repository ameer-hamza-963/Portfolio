Cloudplex

  • Role: Full Stack Developer
  • Tech Stack: React, Node, MongoDB, Express, Elastic Search, AWS, Kibana, Grafana

Description

CloudPlex is a cloud management platform designed to simplify the deployment, management, and scaling of cloud infrastructure. The platform provides businesses with the tools they need to monitor and optimize their cloud resources, ensuring better performance, cost management, and scalability.

Key Features

  • Multi-Cloud Management: Ability to start, manage, and scale cloud servers on AWS, GCP, and Azure from a unified interface.
  • Deployment Logs & Troubleshooting: Displays logs of deployments using Kibana and Elasticsearch, providing deep insights into application performance and errors during deployment.
  • Cloud Resource Monitoring: Real-time monitoring of cloud infrastructure using Grafana and Kiali, enabling users to visualize metrics, system health, and performance.
  • Service Mesh Management: Build and manage Service Mesh from the UI, without the need to navigate external tools. This provides seamless communication and control between microservices in the cloud environment.
  • Live Container Status Monitoring: View live statuses of containers directly from the platform, ensuring that any issues can be quickly detected and addressed.
