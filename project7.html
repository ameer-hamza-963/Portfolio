GoScout

  • Role: Frontend Developer
  • Tech Stack: React, ReduxJS

Description

GoScout is a baseball-focused platform that provides comprehensive match analysis by generating summaries based on player performance and shots played. The platform was already built, and I was responsible for revamping the frontend to enhance user experience and overall functionality.

Key Features

  • Match Summary Generation: Added functionality to automatically generate a detailed summary of each match, highlighting player performance, key moments, and overall team impact.
  • Player Management: Implemented a feature to add new players to the platform, ensuring accurate team rosters and performance tracking.
  • Revamped Frontend UI/UX: Enhanced user experience with a modern and intuitive design.
  • Responsive Design: Optimized for desktop and mobile viewing, ensuring accessibility for coaches, players, and fans.
